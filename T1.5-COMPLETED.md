# T1.5 - 자막 인라인 편집 기능 구현 완료

> **Phase 1, Task 1.5** - 자막 타임라인에서 자막 텍스트를 직접 편집할 수 있는 기능

---

## 구현 내용

### 1. API 엔드포인트

#### PUT /api/subtitles/:id
- 자막 텍스트 수정
- 원본 텍스트 보존 (`originalText`)
- 수정 이력 자동 저장 (`subtitle_edits` 테이블)
- `isEdited: true` 플래그 설정

#### DELETE /api/subtitles/:id
- 자막 삭제 (cascade로 이력도 함께 삭제)

**파일**: `src/app/api/subtitles/[id]/route.ts`

---

### 2. 컴포넌트

#### SubtitleEditor
- 인라인 편집 UI (textarea + 저장/취소 버튼)
- 키보드 단축키:
  - `Enter`: 저장
  - `Shift+Enter`: 줄바꿈
  - `Escape`: 취소
- 자동 포커스 및 텍스트 선택
- 로딩 상태 표시

**파일**: `src/components/subtitle/SubtitleEditor.tsx`

#### SubtitleTimeline (수정)
- 편집 모드 상태 관리 (`editingId`)
- "편집" 버튼 추가
- 더블클릭으로 편집 모드 진입
- 편집 중에는 시크 동작 중단

**파일**: `src/components/subtitle/SubtitleTimeline.tsx`

---

### 3. API 클라이언트

```typescript
import { updateSubtitle, deleteSubtitle, getSubtitles } from '@/lib/api/subtitles';

// 자막 수정
const updatedSubtitle = await updateSubtitle(id, { text: '새 텍스트' });

// 자막 삭제

```

**파일**: `src/lib/api/subtitles.ts`

## 사용 예시

```tsx
'use client';

import { SubtitleTimeline, TimelineSubtitle } from '@/components/subtitle/SubtitleTimeline';
import { updateSubtitle } from '@/lib/api/subtitles';
import { useState } from 'react';

function SubtitlePage() {
  const [subtitles, setSubtitles] = useState<TimelineSubtitle[]>([
    {
      id: '1',
      startTimeMs: 0,
      endTimeMs: 3000,
      text: '안녕하세요',
      isEdited: false,
    },
    {
      id: '2',
      startTimeMs: 3000,
      endTimeMs: 6000,
      text: '경기도의회입니다',
      isEdited: false,
    },
  ]);

  const handleSubtitleUpdate = async (id: string, newText: string) => {
    // 1. API 호출
    const updated = await updateSubtitle(id, { text: newText });

    // 2. 로컬 상태 업데이트
    setSubtitles((prev) =>
      prev.map((s) =>
        s.id === id
          ? { ...s, text: newText, isEdited: true }
          : s
      )
    );

    console.log('자막 수정 완료:', updated);
  };

  return (
    <div className="container mx-auto p-4">
      <h1 className="text-2xl font-bold mb-4">자막 편집</h1>

      <SubtitleTimeline
        subtitles={subtitles}
        currentTimeMs={1500}
        onSeek={(timeMs) => console.log('Seek to:', timeMs)}
        onSubtitleUpdate={handleSubtitleUpdate} // ✅ 편집 기능 활성화
        className="h-96"
      />
    </div>
  );
}

export default SubtitlePage;
```

## 데이터 흐름

```
사용자 클릭 "편집" 버튼
    ↓
SubtitleTimeline: setEditingId(subtitle.id)
    ↓
SubtitleEditor 렌더링 (textarea 표시)
    ↓
사용자 텍스트 수정 후 Enter 또는 "저장" 클릭
    ↓
SubtitleEditor: onSave(id, newText) 호출
    ↓
SubtitleTimeline: handleSave() → onSubtitleUpdate(id, newText)
    ↓
부모 컴포넌트: updateSubtitle(id, { text })
    ↓
API: PUT /api/subtitles/:id
    ├── 기존 자막 조회
    ├── subtitle_edits에 이력 저장
    └── subtitles 업데이트 (isEdited: true, originalText 설정)
    ↓
응답 수신 → 로컬 상태 업데이트
    ↓
SubtitleTimeline: setEditingId(null) → 편집 모드 종료
```
